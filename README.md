# Snake - SFML

This is a remake of the popular console video game "Snake", recreated using C++ and SFML.

Author: Abdulrahman Tolba

# Requirements:  

- MinGW or other g++ compiler <br/>
- SFML 2.4.2 (or above) <br/>
